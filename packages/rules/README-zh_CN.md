### 使用

```
npx @hero-u/rules
```

或全局安装后启动：

```
npm i @hero-u/rules -g
rules
```

进入 http://localhost:7788 管理你的所有 AI 指令：

- 自动检测当前安装的 AI 工具
- 扫描你代码库中存在的指令
- 将现有的指令导入到指令集
- 新增、修改、删除你的指令
- 为所有工具和项目分配合适的指令并同步
